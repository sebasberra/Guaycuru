# refreshEliminarCamas( array $infcamas )

Recorre el arreglo de camas pasado como parámetro y va eliminando de la base centralizada las camas que no estén en dicho arreglo

Recorre el arreglo de camas pasado como parámetro y va eliminando de la base centralizada las camas que no estén en dicho arreglo

Parameters

$infcamas

Es un arreglo del tipo arreglo:

["id_efector"] ID efector donde pertenece la cama integer
["nombre_sala"] Nombre de la sala donde pertenece la cama. NOTA: se busca el id_sala en base central por nombre string
["nombre_habitacion"] Nombre de habitación donde pertenece la cama. NOTA: se busca el id_habitacion en base central por nombre string
["nombre_cama"] Nombre de la cama. NOTA: Se utiliza solo para encontrar la cama, no se modifica string
["id_clasificacion_cama"] Clasificación de camas. Ver tabla: clasificaciones_camas integer
["estado"] L=libre; O=ocupada; F=fuera de servicio; R=en reparación; V=reservada string
["rotativa"] 0=no es rotativa, 1=es rotativa; NOTA: Las camas rotativas pueden cambiarse de habitación o sala o no estar asignada a una habitación en un momento dado integer
["baja"] 0 = habilitada; 1 = baja integer

Returns


Si la función terminó bien no devuelve valor
